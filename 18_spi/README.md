# 18_spi — SPI总线/ICM20608传感器实验

## 1. 目录功能概述

本目录存放I.MX6ULL裸机开发工程第十八个实验——**SPI总线通信与ICM20608传感器驱动**的完整代码。该实验实现I.MX6ULL ECSPI（Enhanced Configurable SPI）控制器的初始化，并通过SPI接口驱动ICM20608六轴运动传感器（三轴加速度计+三轴陀螺仪）。SPI是嵌入式系统中用于高速通信的重要总线，广泛应用于存储器、传感器、显示屏等外设。

## 3. 关键原理简述

### 3.1 SPI总线原理
SPI（Serial Peripheral Interface）总线特点：
- **四线制**：SCK（时钟）、MOSI（主出从入）、MISO（主入从出）、CS（片选）
- **全双工**：同时收发数据
- **主从架构**：一主多从，通过片选信号选择从设备
- **速率**：可达数十MHz

### 3.2 ECSPI寄存器操作
I.MX6ULL的ECSPI控制器通过以下寄存器实现通信：
- **ECSPIx_CONREG**：控制寄存器，配置时钟极性、相位等
- **ECSPIx_CONFIGREG**：配置寄存器，设置片选和burst模式
- **ECSPIx_STATREG**：状态寄存器
- **ECSPIx_TXDATA**：发送数据寄存器
- **ECSPIx_RXDATA**：接收数据寄存器

### 3.3 ICM20608传感器
ICM20608是InvenSense公司的六轴运动传感器：
- **三轴加速度计**：测量X/Y/Z方向加速度
- **三轴陀螺仪**：测量X/Y/Z方向角速度
- **SPI接口**：通过SPI与主控通信
- **数据输出**：16位有符号数表示

通信流程：
1. SPI初始化，配置时钟、极性、相位
2. 向ICM20608写入配置命令（唤醒、量程设置等）
3. 读取WHO_AM_I寄存器确认设备ID
4. 循环读取加速度/陀螺仪数据寄存器

4. 上电启动

## 6. 预期运行现象

开发板上电后：
- LCD屏幕显示ICM20608传感器数据（加速度X/Y/Z、陀螺仪X/Y/Z）
- 晃动开发板时，加速度数据变化
- 旋转开发板时，陀螺仪数据变化

验证标准：传感器数据随运动变化而变化，说明SPI通信正常。

## 7. 常见问题排查

| 故障现象 | 可能原因 | 解决办法 |
|----------|----------|----------|
| SPI通信失败 | 时钟极性/相位不匹配 | 核对ICM20608数据手册中的SPI模式要求 |
| 数据读取全0 | 片选信号未正确控制 | 检查CS引脚配置和控制逻辑 |
| 设备ID读取错误 | 通信未建立 | 检查SPI引脚复用配置和电气连接 |
| 数据跳变异常 | 数据未对齐 | 检查SPI传输数据长度配置 |
