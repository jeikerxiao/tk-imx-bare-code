# 20_pwm_lcdbacklight — PWM背光控制实验

## 1. 目录功能概述

本目录存放I.MX6ULL裸机开发工程第二十个实验——**PWM背光控制**的完整代码。该实验实现PWM（脉宽调制）波形的生成，用于控制LCD屏幕背光亮度。通过调整PWM占空比，可实现背光亮度的无级调节，是嵌入式显示系统中常见的背光控制方案。

## 3. 关键原理简述

### 3.1 PWM原理
PWM（Pulse Width Modulation）通过调节脉冲宽度控制平均电压：
- **周期（Period）**：一个完整PWM波的时间
- **占空比（Duty Cycle）**：高电平时间占周期的百分比
- **平均电压**：`V_avg = V_max × Duty_Cycle`

### 3.2 I.MX6ULL PWM控制器
I.MX6ULL内部集成多个PWM模块，本实验使用PWM1：
- **时钟源**：选择IPG时钟或其他时钟
- **周期寄存器**：设置PWM周期
- **采样寄存器**：设置高电平时间（占空比）
- **极性配置**：设置输出极性

### 3.3 背光控制实现
通过按键或触摸调节占空比：
1. 初始化PWM1，设置周期
2. 设置初始占空比（如50%）
3. 检测按键/触摸输入
4. 根据输入增加或减少占空比
5. 更新采样寄存器，改变背光亮度

## 6. 预期运行现象

开发板上电后：
- LCD屏幕显示背光调节界面
- 通过按键或触摸调节背光亮度
- 背光亮度随占空比变化

验证标准：背光亮度可调，变化平滑，说明PWM控制正常。

## 7. 常见问题排查

| 故障现象 | 可能原因 | 解决办法 |
|----------|----------|----------|
| 背光不亮 | PWM未输出 | 检查PWM引脚复用和使能配置 |
| 背光闪烁 | PWM频率过低 | 提高PWM频率，通常1kHz以上 |
| 亮度调节不连续 | 占空比步进过大 | 减小占空比调节步长 |
| 背光无法调暗 | 占空比下限设置不当 | 设置合理的最低占空比（如5%） |
