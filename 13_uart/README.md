# 13_uart — 串口通信实验

## 1. 目录功能概述

本目录存放I.MX6ULL裸机开发工程第十三个实验——**串口通信（UART）**的完整代码。
该实验实现I.MX6ULL UART1的初始化、数据收发功能，通过串口与PC端串口助手进行双向通信。
串口是嵌入式系统中最常用的调试和通信接口，掌握串口驱动开发是嵌入式工程师的基本功。

## 3. 关键原理简述

### 3.1 UART硬件连接
开发板通过UART1与PC通信，引脚映射：
- **UART1_TXD**：GPIO1_IO16（发送）
- **UART1_RXD**：GPIO1_IO17（接收）
通过USB转TTL模块连接至PC串口助手。

### 3.2 UART配置参数
本实验配置UART1参数：
- **波特率**：115200 bps
- **数据位**：8位
- **停止位**：1位
- **校验位**：无校验
- **流控**：无硬件流控

### 3.3 UART寄存器操作
I.MX6ULL的UART控制器通过以下寄存器实现数据收发：
- **URXD**：接收数据寄存器，读取接收到的字符
- **UTXD**：发送数据寄存器，写入待发送字符
- **UCR1/UCR2/UCR3**：控制寄存器，配置UART工作模式
- **UFCR**：FIFO控制寄存器，配置FIFO深度和触发阈值
- **UBMR/UBIR**：波特率分频寄存器，设置通信波特率
- **USR1/USR2**：状态寄存器，判断发送/接收状态

### 3.4 波特率计算
UART1时钟源为80MHz，波特率计算公式：
```
 Baud Rate = UART_CLK / (16 × (UBMR + 1) / (UBIR + 1))
```
本实验配置为115200 bps，需计算并写入UBMR和UBIR寄存器。

## 5. 下载与验证步骤

1. 编译生成`uart.bin`
2. 使用`imxdownload`工具烧录至SD卡：
```bash
./imxdownload uart.bin /dev/sdX
```
3. SD卡插入开发板，设置SD卡启动模式
4. 使用USB转TTL模块连接开发板UART1至PC
5. PC端打开串口助手（波特率115200，8N1）
6. 上电启动

## 6. 预期运行现象

开发板上电后：
- 串口助手显示提示信息
- 输入字符后，开发板回显相同字符
- LED0随每次字符输入状态翻转

验证标准：串口助手与开发板正常通信，输入字符正确回显。

## 7. 常见问题排查

| 故障现象 | 可能原因 | 解决办法 |
|----------|----------|----------|
| 串口无输出 | 波特率配置错误 | 核对UBMR/UBIR寄存器值，确保波特率为115200 |
| 串口输出乱码 | 时钟源配置错误 | 检查UART时钟源是否为80MHz |
| 串口无响应 | 引脚复用配置错误 | 确认UART1_TXD/RXD引脚已正确复用为UART功能 |
| 接收不到数据 | RXD引脚未配置 | 检查UART接收使能位是否置位 |
| 串口助手连接不上 | USB转TTL接线错误 | 确认TXD/RXD交叉连接，GND共地 |
